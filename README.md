# 🧮 Calculator Project

##  About the Project
This is a simple Calculator web application built using HTML, CSS, and JavaScript.  
It performs basic arithmetic operations like addition, subtraction, multiplication, and division.

---

##  Features
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Clear (C) button
- Responsive design

---

##  Technologies Used
- HTML
- CSS
- JavaScript
